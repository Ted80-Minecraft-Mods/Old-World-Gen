## OLD WORLD GEN ##

The good old minecraft world gen from back in the days
http://www.minecraftforum.net/forums/mapping-and-modding/minecraft-mods/1281910-better-world-generation-4-v1-2-1

Features:
- beta 1.7.3
- alpha 1.2.0
- alpha 1.1.0
- infdev
- indev

Planned:
